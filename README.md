## Sobre o Projeto

O **Rolé App** é uma aplicação mobile que simula uma plataforma social moderna para organização de eventos entre amigos.

A ideia central é funcionar como um **"Waze de Pessoas"**, onde usuários podem:

- criar rolês
- acompanhar presença
- interagir via feed
- gerenciar amizades
- receber notificações em tempo real

O projeto foi desenvolvido com foco em:

- arquitetura escalável
- integração com APIs reais
- persistência de dados
- experiência moderna de usuário

---

## Funcionalidades

### Autenticação
- Login com backend
- Cadastro de usuário
- Persistência de sessão

---

### Rolês (CRUD completo)
- Criar rolê
- Listar rolês
- Editar rolê
- Excluir rolê
- Visualizar detalhes

---

### Presenças (Oracle APEX)
- Adicionar participantes
- Atualizar status
- Remover presença
- Integração com API externa (APEX)

---

### Feed social
- Publicações vinculadas ao rolê
- Comentários por publicação
- Visualização dinâmica
- Interação entre usuários

---

### Amizades
- Enviar pedido de amizade
- Aceitar amizade
- Bloquear usuário
- Lista de conexões
- Simulação de status online/offline

---

### Notificações
- Integração com backend
- Persistência em banco (Neon)
- Exibição na Home
- Atualização automática
- Histórico em modal

---

### Perfil
- Visualização completa
- Edição de dados:
  - Nome
  - Email
  - Telefone
  - Data de nascimento
  - Privacidade
- Persistência no backend
- Atualização automática

---

## Telas do App

| Tela | Descrição |
|------|------------|
| **Splash** | Inicialização e validação de sessão |
| **Login** | Autenticação do usuário |
| **Cadastro** | Registro de novo usuário |
| **Home** | Lista de rolês + notificações |
| **Criar Rolê** | Cadastro de evento |
| **Detalhes** | Informações + feed do rolê |
| **Editar Rolê** | Atualização de evento |
| **Perfil** | Dados do usuário |
| **Editar Perfil** | Atualização de dados |
| **Feed** | Interações sociais |
| **Amizades** | Gestão de conexões |
| **Presenças** | Controle via APEX |
